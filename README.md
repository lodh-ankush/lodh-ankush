## 📌 Featured Projects

### 🎭 VeriFace — AI-Powered Deepfake Detection
> *Deep Learning | Computer Vision | Spatio-Temporal Analysis*

A deep learning system designed to detect manipulated media by analyzing both **spatial (image)** and **temporal (video)** inconsistencies in deepfake content.

- 🏆 Achieved **98.86% Recall** and **F1 Score: 0.947**
- 🧠 Designed a **CNN + LSTM spatio-temporal model** to capture frame-level features and temporal inconsistencies across video sequences
- 📊 Performed advanced evaluation: ROC-AUC, Precision-Recall curves, ablation study, and per-threshold analysis
- 🎯 Identified optimal decision threshold (**0.40**) and key performance factors like batch normalization and dataset scale

`Python` `PyTorch` `OpenCV` `Scikit-learn` `NumPy` `Matplotlib` `CNN` `LSTM` `Deep Learning`

### 📋 LeaveEase — RPA-Based Leave Management System
> *Robotic Process Automation | UiPath | HR Automation*

A simple and efficient tool to **automate and manage employee leave requests** end-to-end using UiPath Studio.

- 🤖 Designed and implemented software robots to handle leave **request submission, validation, approval routing, and status updates**
- ⚙️ Leveraged UiPath workflows, selectors, and activities for accurate data handling and exception management
- ⏱️ Significantly reduced manual HR intervention and improved turnaround time

`UiPath Studio` `Visual Basic .NET` `C#` `RPA` `HR Automation`
